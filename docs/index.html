Analysis of the Effects of Economic Growth on Health Outcomes

Project Overview

This project investigates the relationship between economic indicators and health outcomes across different countries from 2000 to 2017.

Key Findings

  • Direct correlation between GDP per capita and improvements in life expectancy.
  • Significant variations in health metrics like IMR and MMR across different income levels.

Tools Used

  • R and RStudio
  • ggplot2 for data visualization
  • Tableau for additional insights

How to Run the Code

Clone the repository and open the Analysis.R script in RStudio. Install required packages using the following R command:

install.packages(c("ggplot2", "dplyr", "tidyr"))
